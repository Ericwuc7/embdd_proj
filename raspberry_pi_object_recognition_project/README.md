# Raspberry Pi Object Recognition Project

This folder is reserved for your Raspberry Pi object recognition project.

## Goal
- Use a trained `.pt` model
- Run inference on Raspberry Pi camera input
- Recognize objects in real time

## Suggested next steps
- Add your model file in this folder (or a subfolder like `models/`)
- Create your Python inference script (for example `main.py`)
- Add a `requirements.txt` for dependencies
