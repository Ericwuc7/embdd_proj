# Hardware Connections

## Arduino Uno Wiring

### PIR Sensor
- VCC -> 5V
- GND -> GND
- OUT -> D8

### Buzzer
- + -> D9
- - -> GND

### LED
- Anode -> D10 (through 220Ω resistor)
- Cathode -> GND

## Raspberry Pi 3B+
- Arduino Uno connected by USB cable
- USB camera connected to USB port
- Raspberry Pi connected to Wi-Fi for Telegram/dashboard access
