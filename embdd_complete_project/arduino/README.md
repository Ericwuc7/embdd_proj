# Arduino Firmware

## Files
- `pir_motion_detection.ino`: PIR-only serial motion events.
- `pir_with_alarm.ino`: PIR + buzzer + LED + serial events.

## Board and Port
- Board: Arduino Uno
- Baud rate: 9600

## Upload Steps
1. Open Arduino IDE.
2. Open one of the `.ino` files.
3. Select Board/Port.
4. Upload.
